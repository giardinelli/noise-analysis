# noise-analysis

MATLAB live scripts for noise analysis (analytical and synthetic, stages 1–2).

[![Open in MATLAB Online](https://www.mathworks.com/images/responsive/global/open-in-matlab-online.svg)](https://matlab.mathworks.com/open/github/v1?repo=giardinelli/noise-analysis)

Click the button above to open and run these live scripts directly in **MATLAB Online** — no download needed.

## Files

| File | |
|------|--|
| `stage1_analytical.mlx` | Stage 1 — analytical |
| `stage1_5_analytical_noise.mlx` | Stage 1.5 — analytical noise |
| `stage1_75_noise_background.mlx` | Stage 1.75 — noise background |
| `stage1_99_autoF_everything.mlx` | Stage 1.99 — auto-F, everything |
| `stage2_synthetic_noise.mlx` | Stage 2 — synthetic noise |
| `stage2_SVD_synthetic_noice.mlx` | Stage 2 — SVD synthetic noise |

> Note: `.mlx` live scripts are a binary format and don't preview as text on GitHub. Use the **Open in MATLAB Online** button to view them rendered, or see the exported `.m` versions (plain text) if present.
